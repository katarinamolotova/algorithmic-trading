О программе

Algorithmic Trading

  • Данные загружаются в программу из файла с котировками акций с расширением .csv
  • При загрузке нового файла с данными поле для рисования графиков очищается
  • Пользователем задаётся количество точек, по которым должен быть построен график (количество точек не меньше, чем в загруженном файле)
  • Все точки распределены равномерно между начальной и конечной датами
  • На итоговом графике соседние точки соединены прямой линией

  • Реализован метод интерполяции кубическим сплайном
  • Реализовано построение графиков таблично заданной функции котировок акций с использованием метода наименьших квадратов

  • В интерфейсе есть кнопка для отрисовки графика, полученного кубическим сплайном
  • В интерфейсе есть кнопка очистки поля для рисования графиков
  • На поле может быть одновременно изображено до 5 графиков (если на поле уже отображено 5 графиков, блокируются отрисовка новых графиков)
  • В интерфейсе указана следующая информация о графиках (цвет, имя файлов, для аппроксимации степень полинома)
  • Предусмотрена возможность по заданному пользователем значению аргумента вывести значения функции котировок акций, полученные методом интерполяции

    • Установка

    Для того, чтобы начать работу с программой нужно использовать команду make install.

    Удаление

    Для удаления программы вы можете использовать команду make uninstall.

    Перемещение

    Для переноса программы в формате архива необходимо использовать команду make dist.

    Очистка

    Для очистки директории необходимо использовать команду make clean.
